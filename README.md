## Scoring methodology

Each creator scored **1–10 across 6 criteria**, weighted into a single **Overall Score (0–100)**:

| Criterion | Weight | Why |
|---|---:|---|
| **Engagement rate** | **20%** | Raw followers without engagement is dead reach. |
| **Cross-platform presence** | **20%** | Single-platform creators are fragile (TikTok ban risk, algorithm shifts). |
| Content quality | 18% | The long-term moat. |
| Follower count | 18% | Distribution still matters — but not everything. |
| Commercial influence | 16% | Brand deals, owned products, monetisation. |
| Posting frequency | 8% | Quality > quantity. |

## Three sortable columns

- **Overall** — composite weighted score (0-100)
- **Followers** — raw follower count (across primary platform)
- **Engagement** — engagement-rate score (×10, so 8.2% = 82)

Sort by **Engagement** to see who actually moves audiences (Veritasium, Karen X. Cheng, IShowSpeed). Sort by **Followers** for raw reach (Cristiano Ronaldo at 660M, Selena Gomez at 425M, MrBeast at 425M). Sort by **Overall** for the balanced view.
